参观现场

发表于 2022-08-20 分类于 生活

今天上午依然是学习文件。而下午我们要进行现场的参观和培训，以及培训后的考察。

我们参观现场需要爬上38米高的桥梁，虽说梯子是走的，不是塔吊那种直梯，而且周围有防护网，但依然害怕的要命。天也很热，爬上去，一身的汗。下去时，依然很害怕，上去时很少看下面，而下去的话，看下面会比较多。还好顺利地完成了上下，没有摔倒、晕倒。不知道以后上的频繁不，据说还要爬塔吊，害怕！

参观回来，进行了《全员安全生产责任制度》的培训，以及考试，最后得到了90分的成绩，还好吧，一个多选题做错了，扣了10分。

来到项目的第二天，也是被教育的一天

发表于 2022-08-19 更新于 2022-08-20 分类于 生活

今天，召开了新员工见面会，依然少不了自我介绍，不过还好，参会人数不多，没有华东公司组织的培训那次那么紧张。然后就是入场安全教育。整个上午和下午都是在让自己看安全手册这些，中午没休息好，下午困得要死。

标题之所以称今天是被教育的一天，是因为中午吃饭被一位前辈说男人要有自信，我吃饭看起来就很没有自信，很女孩子气。还有翘兰花指……其实并非兰花指啦，只是我不会用筷子，小拇指翘起来，很像兰花指而已。这些更多是因为社恐的我，来到新环境，身边没有一个认识的，很紧张的缘故，往后会好起来吧。

到达项目部的第一天

发表于 2022-08-18 更新于 2022-08-24 分类于 生活

傍晚到达了项目部，领导们、师兄们都已经在拉好条幅等待我们。

合影后，吃了晚饭，就自由休息了。吃饭真的把我热了一身的汗，本身就容易出汗，加上空调温度设置的比较高，应该还是刚开的。今天就写到这吧，没多少东西要写，而且很晚了，困倦。
